Updates

The UI of the main menu has been revamped. It is no longer a multiplayer game. Its now a solo player in which you play against the AI. The sound effects are implemeented along with an arcade style type of music. The commands in the options are fully functionable. You are able to pause the game with command and are able to quit anytime soon. The game does not save your progress, however, since this game does not have a story tied to it . This should be an issue.
